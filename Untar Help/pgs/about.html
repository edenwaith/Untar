Untar Help

Learn about Untar

Untar is a graphical interface program, which wraps itself around UNIX command-line utilities such as 7za, tar, uncompress, bzip2, unrar, and unzip. Untar can uncompress and extract files from archives. It supports the following files: .7z, .tar, .tar.gz, .tgz, .tar.z, .z, .gz, .svgz .Z, .tar.Z, .taz, .tar.bz, .tar.bz2, .tbz, .tbz2, .bz, .bz2, .rar, and .zip.

Standard Use

Launch Untar and then select the file you wish to uncompress or unarchive, and click on the Untar button. For some files, such as .7z, .tar, .tgz, .rar, or .zip, they can be placed in a user-specified location by setting the Directory field.

Drag and Drop

Drag and drop a compressed or archived file onto the Untar application icon, and Untar will launch automatically and uncompress the file.
